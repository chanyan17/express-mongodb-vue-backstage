# backend

## [后台开发参考](https://zhuanlan.zhihu.com/p/44685298)

## 依赖包

* express

* mongoose

* nodemon
    ```bash
    
    npm install --save-dev nodemon

    ```

    修改`package.json`文件的`scripts`

    ```javascript

    "scripts": {
        "start": "nodemon app.js"
    },

    ```

    启动express服务

    ```bash

    npm run start

  ```
